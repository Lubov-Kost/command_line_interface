# command_line_interface

Решение задач с Rosalind к семинару **"Интерфейс командной строки. Исключительные ситуации"**

## Тут есть

Файлы основных программ:
  - `genetic_code.py` для первой задачи "Translating RNA into Protein"
  - `haystack.py` для второй задачи "Finding a Motif in DNA"
  - `common_ancestor.py` для третьей задачи "Consensus and Profile"
  - `components.py` для четвертой задачи "Connected Components"

А также примеры файлов для 3 и 4 задач: `fasta.txt` и `graph.txt`

## Примеры запуска через командную строку

```bash
python genetic_code.py AUGGCCAUGGCGCCCAGAACUGAGAUCAAUAGUACCCGUAUUAACGGGUGA
```

```bash
python haystack.py GATATATGCATATACTT ATAT
```

```bash
python common_ancestor.py fasta.txt
```

```bash
python components.py graph.txt
```
